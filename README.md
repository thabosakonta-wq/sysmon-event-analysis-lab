Sysmon Event Analysis Lab
A cybersecurity project focused on analyzing Sysmon events, detecting suspicious activity, mapping detections to MITRE ATT&CK techniques, and documenting investigation findings.

Overview
This project demonstrates how SOC analysts investigate endpoint telemetry generated by Sysmon. Detection scripts identify suspicious behavior such as PowerShell execution, process creation activity, and outbound network communications.

Features
PowerShell Detection
Detects PowerShell execution activity
Identifies potentially suspicious command execution
Maps to ATT&CK T1059.001
Process Creation Detection
Detects Sysmon Event ID 1
Monitors process execution activity
Supports endpoint investigation workflows
Network Connection Detection
Detects Sysmon Event ID 3
Identifies outbound network communications
Supports threat hunting activities
Investigation Reporting
Documents findings
Records severity assessments
Maps detections to MITRE ATT&CK
